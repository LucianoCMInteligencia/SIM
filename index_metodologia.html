Procedimiento de Investigación

El procedimiento seguido en esta investigación se desarrolló de manera secuencial y organizada, integrando distintas etapas que permitieron obtener información válida y coherente con los objetivos planteados. Cada paso fue realizado de forma consciente, combinando el rigor académico con una comprensión personal del proceso investigativo.

  1. Revisión inicial del tema y del mercado tecnológico: Se realizó una exploración preliminar para comprender el contexto general y las tendencias actuales relacionadas con el consumo de productos tecnológicos.

    Punto 1. Revisión inicial del tema

  2. Definición del problema y de los objetivos del estudio: Se estableció con claridad el propósito de la investigación y las preguntas que guiaron el análisis.

    Punto 2. Definición del problema y objetivos

  3. Selección del enfoque y tipos de investigación: Se adoptó una combinación de enfoques exploratorio, descriptivo, correlacional y explicativo para abordar el fenómeno desde distintas perspectivas.

    Punto 3. Selección del enfoque y tipos de investigación

  4. Identificación de la población y selección de la muestra: Se definió como población a los consumidores de productos tecnológicos y se seleccionó una muestra no probabilística por conveniencia.

    Punto 4. Identificación de la población y selección de la muestra

  5. Diseño de los instrumentos de recolección: Se elaboraron el cuestionario para la encuesta, el guion para las entrevistas, la lista de cotejo para la observación y las fichas de análisis para la revisión documental.

    Punto 5. Diseño de los instrumentos de recolección

  6. Aplicación de las técnicas de recolección: Se llevaron a cabo encuestas, entrevistas, observaciones directas y revisión de documentos relacionados con el mercado tecnológico.

    Punto 6. Aplicación de las técnicas de recolección

  7. Registro y organización de los datos obtenidos: La información recolectada fue clasificada y ordenada para facilitar su análisis posterior.

    Punto 7. Registro y organización de los datos obtenidos

  8. Preparación para el análisis: Se estructuraron los datos y se definieron los criterios analíticos que permitirían interpretar los resultados de manera coherente.

    Punto 8. Preparación para el análisis

Este procedimiento no solo permitió obtener datos relevantes, sino también comprender de manera más profunda el proceso investigativo. Cada etapa aportó una visión distinta y complementaria, reforzando la importancia de un enfoque metodológico consciente y bien fundamentado.

Métodos de Análisis de Datos

Para interpretar la información obtenida en esta investigación se empleó un enfoque de análisis mixto, combinando procedimientos cuantitativos y cualitativos. Esta integración permitió comprender tanto los patrones numéricos presentes en las encuestas como los significados y percepciones expresados en las entrevistas, la observación y la revisión documental.

Análisis Cuantitativo

El análisis cuantitativo se aplicó a los datos obtenidos mediante la encuesta. Se utilizaron técnicas descriptivas básicas que permitieron identificar tendencias generales y comportamientos de la muestra. Entre los procedimientos empleados se incluyen:

  1. Cálculo de frecuencias: para determinar la cantidad de respuestas por categoría.
  2. Porcentajes: para expresar la distribución de las respuestas de manera proporcional.
  3. Tablas y gráficos: para representar visualmente los resultados y facilitar su interpretación.
  4. Comparación entre variables: para explorar posibles relaciones entre las respuestas.

Análisis Cualitativo

El análisis cualitativo se aplicó a la información obtenida mediante entrevistas, observación directa y revisión documental. Este proceso permitió identificar patrones, significados y categorías relevantes dentro del discurso y del comportamiento observado. Las técnicas utilizadas fueron:

  1. Codificación abierta: identificación inicial de ideas, conceptos y fragmentos relevantes.
  2. Categorización: agrupación de códigos en temas o dimensiones significativas.
  3. Análisis temático: interpretación de los temas emergentes para comprender percepciones y tendencias.
  4. Triangulación: comparación entre los datos de distintas técnicas para aumentar la validez del análisis.

La combinación de ambos enfoques permitió obtener una visión más completa del fenómeno estudiado, integrando tanto la dimensión numérica como la interpretativa de los datos.
